# PRISM Media Player

## Purpose
- PRISM Media Player is meant to serve as a tool for managing and interfacing with media present on a local server i.e. a home server.

![PRISM Media Player: UI](/assets/images/readme-images/prism-media-player.png)  

## Features
- [x] Load media through scan of chosen directory
- [x] Search based on Artist, Album, Track Title, etc
- [x] Play, Pause, Skip Forward, Skip Back functionality
- [x] Add / Remove songs from queue
- [ ] Rearrange queue

- More features to be determined
