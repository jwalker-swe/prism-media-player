# PRISM Media Player

## Purpose
- PRISM Media Player is meant to serve as a tool for managing and interfacing with media present on a local server i.e. a home server.

## Features
- [ ] Load media through scan of chosen directory
- [ ] Search based on Artist, Album, Track Title, etc
- [ ] Play, Pause, Skip Forward, Skip Back functionality
- [ ] Add / Remove songs from queue
- [ ] Rearrange queue

- More features to be determined
